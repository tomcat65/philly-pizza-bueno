# PhillyPizzaBueno

Sistema de gestión y control de operaciones para negocios de pizzas.

## Stack Tecnológico

- React
- Next.js 14
- TypeScript
- TailwindCSS
- Supabase
- Vercel
- Shadcn/UI

## Características

- Manual de procedimientos interactivo
- Catálogo de ingredientes con detalles
- Guía de preparación de productos
- Interfaz moderna y responsiva
- Navegación optimizada

## Desarrollo Local

```bash
# Clonar el repositorio
git clone https://github.com/yourusername/philly-pizza-bueno.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## Despliegue

La aplicación está desplegada en Vercel y se actualiza automáticamente con cada push a la rama main.
